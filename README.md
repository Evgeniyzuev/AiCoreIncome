# 

This repository contains the initial setup and the final version of the  project, including the basic project structure, necessary images, and icons.

## Getting Started

To get started with either the initial setup or the final version, follow these instructions:

### Cloning the Repository

1. **Clone the Repository**:

    ```bash
    git clone 
    cd 
    ```

### Initial Setup

2. **Switch to the `initial-setup` Branch**:

    ```bash
    git checkout initial-setup
    ```

3. **Install Dependencies**:

    ```bash
    npm install
    ```

4. **Run the Development Server**:

    ```bash
    npm run dev
    ```

5. **Open Your Browser**:

    Navigate to localhost to see the initial setup.

### Final Version

2. **Switch to the `final-version` Branch**:

    ```bash
    git checkout final-version
    ```

3. **Install Dependencies**:

    ```bash
    npm install
    ```

4. **Run the Development Server**:

    ```bash
    npm run dev
    ```

5. **Open Your Browser**:

    Navigate to localhost to see the final version of the application.
